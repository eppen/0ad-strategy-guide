# 入门指南

## 下载与安装

1. 访问 [play0ad.com](https://play0ad.com/) 下载 Windows / Linux / macOS 安装包。
2. 游戏完全免费：无内购、无广告。代码 GPL v2，美术 CC BY-SA 3.0。
3. **更新前关闭所有 Mod**，避免版本冲突。
4. Release 28 起提供 Windows 64 位版；旧版 Windows 7/8.1 已不再支持。

### 推荐配置

- CPU：3 GHz 及以上
- 内存：4 GB+（64 位版更稳）
- 显卡：支持 OpenGL 2.1+；可选 Vulkan 后端提升帧率

## 首次启动

### 主菜单模式

| 模式 | 说明 |
|------|------|
| Skirmish（遭遇战） | 标准资源开局，对抗 AI 或空位 |
| Deathmatch | 高资源快攻，适合练兵种 |
| Capture the Relic | 争夺圣物，持有时长获胜 |
| Multiplayer | 大厅联机或 IP 直连 / LAN |
| Campaign / Scenario | 官方或社区剧本（随版本而异） |

新手建议：**Skirmish → 1v1 → 标准资源 → AI 难度 Normal**。

### 地图与文明选择

- **小地图**（128–256）适合练经济；大地图重后期与海军。
- 首局推荐 **罗马**（兵种均衡）、**孔雀**（象兵直观）或 **雅典**（经济+海军）。
- 可勾选「Reveal Map」观战学习 AI 建造顺序。

## 界面要点

- **小地图**：右键移动视角；注意敌方颜色与侦察范围。
- **选择组**：`Ctrl+数字` 编组；公民兵与军事单位分开编组。
- **集结点（Rally Point）**： Civic Center、Barracks 上设置，新兵自动去采集或前线。
- **阵型**：选中军事单位后使用 Phalanx、Box 等；密集阵型利于近战，怕溅射与攻城。
- **阶段按钮**： Civic Center 顶部显示当前阶段及升级进度。

## 第一局 15 分钟 checklist

```
0:00  分配 2 平民采浆果，2 砍树，其余待 Civic Center 产出
0:30  公民兵协助砍树或探图
2:00  浆果旁建 Farmstead；树林旁建 Storehouse
4:00  补 House 至人口上限；开始猎鹿或养羊
6:00  第二 Civic Center 或扩张采石/金属（视地图）
8:00  满足 5 建筑 → 研究 Town Phase
9:00  建 1–2 座 Sentry Tower；Barracks 产矛兵
12:00  采石采矿，准备 City 资源
15:00  小股部队试探 AI，不要裸 Civic Center
```

## 常见新手错误

1. **平民走太远采集**——不建 dropsite，效率腰斩。
2. **过早全力造兵**——Town 前经济崩则无力防守。
3. **单位散开**——被各个击破；进攻防守都要保持编队。
4. **忽视领土**——建筑须在领土内（码头、前哨除外）；领土由 Civic Center 等扩张。
5. **只造一种兵**——弓兵怕骑兵，矛兵怕弓兵，需混合或针对性出兵。
