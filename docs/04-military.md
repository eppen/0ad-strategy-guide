# 军事战术

## 兵种职能

| 类型 | 作用 | 注意 |
|------|------|------|
| 矛兵 / Hoplite | 抗骑兵、前线 | 怕远程风筝 |
| 剑兵 | 通用近战 | 成本较高 |
| 弓兵 / 投石 | 远程输出 | 怕骑兵突脸 |
| 轻骑兵 | 骚扰、追射、占线 | 避免硬撞矛阵 |
| 重骑兵 / 冠军骑 | 冲散阵型 | R28 冠军骑 HP 下调 |
| 攻城器 | 拆墙拆 CC | 需步兵保护 |
| 象兵 | 高 crush、溅射 | 怕集中 pierce、长矛 |
| 船 | 控海、登陆 | 分专精舰种 |

## 阵型

- **Phalanx / Line**：增加前排护甲，适合 Hoplite、矛兵抗骑。
- **Box**：四面防御，守点、护攻城器。
- **散阵**：对投石溅射略好，但失去阵型加成。

**原则**：接战前编队；切忌长途奔袭后散乱接战。

## 防守

1. **墙 + 塔**：拖延时间，便于调兵；斯巴达无石墙，用部队填缺口。
2. **驻军在 CC / Fortress**：增加箭矢与占领抗性。
3. **见敌集结**：全民兵（男性公民兵）拉回，平民躲二线或进 CC。
4. **分线**：小股骑兵骚扰敌方经济，迫其分兵。

Release 28 后 CC/Fortress 更难占，但 **断经济**（砍 dropsite、杀 Trader）仍有效。

## 进攻

### 标准攻城序列

```
侦察 → 集结（步兵 + 骑 + _ram/catapult）→ 墙口突破
→ 攻城器在后、Ram 在前 → 步兵清兵 → 优先 CC / Barracks / Market
```

- **Battle Ram** 吸收塔伤；远程 siege 藏后排。
- 部分骑兵 **佯攻** 引守军离开，主力打另一侧——注意家里 CC 别空。

### 目标优先级

1. Civic Center（无法训练平民/升阶段）
2. Barracks / Stable / 特殊兵营
3. dropsite 与农田
4. Fortress、Wonder

### 对 AI 的稳定套路

- Town 后 **小波次** 试探，摸清 AI 兵种。
- City 后 **混合部队 + 2+ 攻城器** 推 CC；象兵文明可大象顶前排（见孔雀篇）。

## 海战要点（R28）

- 早造 **scout ship** 控视野。
- **Fire ship** 造价降低，适合伏击；**Arrow ship** 加强，远程对射主力。
- **Ram ship** 伤害下调，需保护接近。
- 海军科技树简化——按版本查看 Wiki 当前节点。

## 克制关系（简化）

```
矛兵 ──→ 骑兵 ──→ 弓兵/投石 ──→ 步兵 ──→ ...
攻城器 ──→ 建筑；需步兵防骑
象兵 ──→ 建筑/密集步兵；怕高 pierce 集火
```

具体数值因文明科技而异，**开自定义对战测试** 最可靠。

## 微操技巧

- **Focus fire**：右键优先集火低血或 siege。
- **Kiting**：弓兵/骑射 hit-and-run。
- **Garrison**：远程兵进塔/CC 增伤； Minister（汉）不再提供 garrison 战斗加成（R28）。
- **Split**：两路进攻迫使分兵；需 APM 与地图控制。

## formation 与 Group Movement

R28 改进 **群组移动**：终点附近自动散开，减少一字长蛇。大部队仍建议分编组进场。
