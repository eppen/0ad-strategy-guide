# 进阶技巧

## 公民兵经验管理

- 交战前用 **低经验公民兵** 采集，保留 **高 rank** 守家或决战。
- 战后不要把残血高 rank 兵送回砍树——效率低且易被偷袭。

## 地图读取

开局 1 分钟内确认：

- 金属矿距离（决定 City 时间）
- 海/湖（Dock 计入 5 建筑、海军线）
- 邻居方位（rush 方向）
- 鹿群与浆果位置（Village 食物上限）

## 假进攻与牵制

- 小股骑兵扰 dropsite → 对手分兵 → 主力攻另一侧。
- 代价：自己经济也暂停；计算人口与 dropsite 价值。

## 对特定单位的反制

| 敌方 | 应对 |
|------|------|
| 战象 | 高 pierce 矛/弓集火；不要散兵 |
| 重骑 | 矛阵 + 墙 choke |
| 骑射 | 轻骑追或弓兵数量压制 |
| Siege | 骑兵绕后或步兵快速贴脸 |
| Fire ship | 勿密集舰队；scout 探路 |

## AI 难度

- **Sandbox**：练建造顺序。
- **Normal**：本攻略默认参照。
- **Hard / Very Hard**：AI 加成与多线进攻；需更早 Town 塔与 City siege。

## Mod 推荐（非官方）

| Mod | 说明 |
|-----|------|
| Delenda Est | 大改平衡与新 faction，社区热门 |
| Millennium A.D. | 中世纪主题 total conversion |
| Terra Magna | 增加 Scythian 等文明 |

Mod 与原版平衡无关；玩 Mod 前阅读说明并 **更新前禁用 Mod**。

## Atlas 地图编辑器

- 自制历史战役（如高卢战争）；教育向用法见 Wiki **Atlas** 条目。
- 可导入高度图模拟真实地形。

## 性能

- Release 28：**64 位**、Vulkan、FSR；低配可降特效。
- 大规模战斗卡顿时：减少同时选中的单位数、关部分粒子。
